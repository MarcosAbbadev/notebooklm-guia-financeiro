# Guia Financeiro - NotebookLM
Este projeto aplica o **NotebookLM** como ferramenta de **Inteligência Artificial Generativa voltada à educação financeira**. A proposta é **estruturar**, **sintetizar** e **explicar** conceitos essenciais relacionados à saúde financeira, ao planejamento mensal e às decisões financeiras de curto prazo. Todo o conteúdo produzido é fundamentado **exclusivamente em fontes teóricas previamente definida**s, garantindo consistência e evitando informações incorretas.

O NotebookLM desenvolvido neste projeto pode ser acessado por meio do link abaixo:

https://notebooklm.google.com/notebook/43d13da4-4b6e-418b-958b-4ec1a06f77b3

___

## Objetivo do Projeto

- Criação de um caderno temático no NotebookLM sobre um assunto financeiro introdutório.
- Definir objetivos de estudo
- Elaborar perguntas estratégicas
- Testar variações de scripts

___

## Funcionalidades do NotebookLM

- Resumos conceituais
- Mapas mentais
- Relatórios explicativos
- Cartões didáticos
- Testes conceituais
- Infográficos
- Resumo em vídeo

❗ Para testar o notebookLM, é recomendado o uso de algumas das alternativas acima ❗

## Fontes

## Demonstração do Funcionamento

### Exemplos de uso do NotebookLM, incluindo:

- **Interações Individuais:** Exemplos de prompts focados em dúvidas comportamentais, como compras por impulso.
- **Resumo:** Explicações textuais estruturadas.
- **Resumo em Áudio:** Resumo gerado em formato de áudio para consumo rápido.
- **Vídeo:** Demonstrações visuais de conceitos de saúde financeira.
- **Avaliações e Testes:** Questionários de múltipla escolha gerados automaticamente para fixação de conteúdo.

Para uma experiência detalhada com capturas de tela, áudios e vídeos, acesse a documentação completa em: **[/demonstracao](./demonstracao)**.
